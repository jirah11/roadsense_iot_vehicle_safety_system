# roadsense_iot_vehicle_safety_system

Project Title: RoadSense: IoT-Based Vehicle Environmental Safety System

Description: RoadSense is a mobile application Vehicle Safety Monitoring System designed to provide drivers and administrators with real-time information about environmental hazards affecting their vehicles.

Technologies Used: Flutter, Dart, IoT

Features: 
Dashboard / Home Module:
- Overview of vehicle status at a glance
- Safety indicator (Safe / Caution / Danger)
- Summary of current environmental readings (flood level, temperature)
- Quick access to alerts and logs
Real-Time Alerts Module:
- Displays active alerts generated by the IoT sensors
- Visual and textual warnings
- Optional: Color-coded severity indicators (Green, Yellow, Red)
- Option to acknowledge alerts
Vehicle Profile Module:
- Allows the user to select their vehicle type:
- Sedan, SUV, Hatchback, Cupe
- Adjusts system thresholds for flood and temperature detection
- Shows basic vehicle info (such as nickname and plate number)
Historical Data / Logs Module:
- Displays past alerts and safety events
- Timestamped logs for reference
- Graphs or charts showing trends in flood/temperature
Settings Module:
- Adjust notification preferences (sound, vibration, alert types)
- Configure IoT device connection (e.g., Wi-Fi credentials)
- Set preferred units (°C/°F, cm/inches)
- Option to reset app data or synchronize with cloud
Help / Information Module:
- Short guide on how the system works
- Explanation of alert meanings and safety thresholds
- Tips for safe driving in floods and extreme heat
IoT Device Status Module:
- Shows connection status of the vehicle IoT module
- Sensor readings in real-time (raw values)
- Indicator if any sensor is disconnected or malfunctioning
